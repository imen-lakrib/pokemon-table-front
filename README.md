# Pokémon Table App

This is a React Vite.js web application that displays a table of Pokémon with pagination and filtering capabilities using the Material-UI (MUI) library.


## Features

- Display a table of Pokémon data.
- Paginate through Pokémon data.
- Filter Pokémon data by name and power.
- Responsive design for various screen sizes.

## Technologies Used

- [React](https://reactjs.org/)
- [Vite.js](https://vitejs.dev/)
- [Material-UI](https://mui.com/)
- [axios](https://axios-http.com/)

## Getting Started

### Prerequisites

- Node.js and npm/yarn installed on your machine.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/imen-lakrib/pokemon-table.git
   ```

2. Navigate to the project directory:

   ```bash
   cd pokemon-table-app
   ```

3. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

### Usage

1. Start the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   ```

2. Open your browser and visit [http://localhost:5173](http://localhost:5173) to see the Pokémon table app in action.

### Folder Structure

- `src/` contains the source code for the application.
- `public/` contains static assets.

### Customization

You can customize this application by modifying the following files:

- `src/components/PokemonTable.jsx`: Customize the appearance and behavior of the Pokémon table.
- `src/utils/data.js`: Modify the Pokémon data or table column configuration.
- Add your own styles in `src/styles` if needed.
